# ISO19650 Knowledge Maps

https://ibrahimfahdah.github.io/iso19650-knowledge-maps/

ISO19650 Knowledge Maps was created to help people understand the ISO 19650 standard through interactive visual maps.

ISO 19650 contains many connected ideas such as roles, processes, information requirements, systems, and responsibilities. Reading the standard in a linear way can make these relationships hard to see. 

This website presents those relationships as interactive knowledge maps so users can explore how concepts connect and how different parts of the standard relate to each other.

## Why This Was Created

This website was created to make ISO 19650 (2018) and the draft version (2026) easier to learn, explain, compare, and discuss.

It helps users:

- understand the structure of ISO 19650 visually
- see how roles, processes, and information concepts connect
- explore relationships between concepts more quickly
- inspect supporting evidence and references behind those relationships
- compare maps across different versions of the standard

## Who It Is For

This site can be useful for:

- BIM managers
- information managers
- project delivery teams
- asset owners
- digital construction professionals
- students and researchers
- anyone learning ISO 19650

## What You Can Do

With ISO19650 Knowledge Maps, you can:

- choose a standard version
- choose a knowledge map
- explore the graph visually
- zoom and pan across the map
- click nodes to inspect concepts, roles, systems, and processes
- click edges to inspect relationships, evidence, and source references

## How To Use

1. Select the ISO 19650 version from the sidebar.
2. Select a knowledge map.
3. Explore the graph in the main viewer.
4. Click a node to see its details.
5. Click an edge to see relationship evidence and references.
6. Use the Legend to understand the color coding.

## What The Colors Mean

The maps use colors to distinguish different types of nodes:

- Blue: Concept
- Green: Role
- Orange: Process
- Purple: Information
- Gray: System

## About The Knowledge Maps

The purpose of the maps is not to replace the standard, but to support learning, interpretation, communication, and exploration.

## Important Note

This website is an educational and interpretive visualization tool.

For formal, contractual, or authoritative use, always refer to the official ISO 19650 publications and related guidance documents.
